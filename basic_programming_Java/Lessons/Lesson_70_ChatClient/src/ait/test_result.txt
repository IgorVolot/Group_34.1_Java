Это все проблемы сервера.

1. Сервер реагирует на изменение кол-ва клиентов только спустя одно событие (сообщение) от клиента. Возможно это не баг.
2. Имя клиента не проверяется на уникальность.
3. Имя клиента может быть пустой строкой (null???).
4. Не возможно выйти из клиента (чата) пока не введешь свое имя или пустую строку вместо имени.
5. Клиент допускает как имя команду (слово) для выхода.
